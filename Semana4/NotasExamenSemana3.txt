En el archivo pdf de nombre "Examen semana 4 Manuel Mauricio Chulim Alamilla"
se encuentran las preguntas del examen. 

En cuanto a los proyectos, se encuentran en las siguientes carpetas:

-El proyecto que expone un servicio rest se encuentra en la carpeta de nombre
RestApi&AppMvc.

-El proyecto que usa un cliente que consume el servicio rest, se encuentra en
la carpeta crm-web-app-rest-client-demo.

-El proyecto que usa spring batch para generar una tabla en una base de datos
de MySQL a partir de un archivo csv, se encuentra en la carpeta spring-batch.
